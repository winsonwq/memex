## Memory Palace 自我实现 — Phase 0 完成

### 项目启动

阿秋提出「开始自我实现，有阶段性的提交，和记录」。

**目标**：构建 AI 的记忆能力，让 AI 能在跨会话、跨项目中持续学习和记忆。

**核心理念**（来自 MemPalace 调研）：
- Verbatim 原文存储，不依赖 LLM 提取
- 宫殿结构组织（wing/room/closet/drawer）
- 96.6% R@5 raw baseline 证明简单方法比复杂方法更强
